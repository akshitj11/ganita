# ganita / गणित
```
बुद्धिगुणा एव बीजं च।

निर्मलत्वं च बुद्धेः अवगम्यते॥
```

```
The attributes of the intellect are the true seed of analysis;
the spotless clarity of the mind is where true algebra resides.
```
